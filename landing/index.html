What makes Schism different

🎯

Visual Sliders

Drag sliders to adjust creativity, humor, formality, and more. See changes in real-time. No code required.

🧠

SAE-Powered Steering

Uses Sparse Autoencoders to identify interpretable features inside the model. Based on cutting-edge 2025 research (FGAA).

💻

100% Local

Everything runs on your machine. No cloud, no API keys, no data leaving your laptop. Works on Apple Silicon.

⚡

Zero Training

No fine-tuning, no LoRA, no RLHF. Schism steers behavior at inference time by manipulating internal activations.

🎭

Presets

Ship with 11 built-in personality presets. Create your own, export as JSON, share with the community.

🔍

Compare Mode

See steered vs vanilla output side-by-side. Understand exactly how your sliders change the model's behavior.

How it works

1

Install and download a model

pip install schism then schism download gemma-2-2b to grab a small model (~5GB) and its pre-trained Sparse Autoencoder.

2

Launch the UI

Run schism and a browser window opens with personality sliders. Each slider maps to a steering vector computed from contrastive prompt pairs.

3

Steer the model

Adjust sliders and type a prompt. Schism injects steering vectors into the model's hidden states during inference, reshaping its behavior without changing any weights.

4

Save and share

Found a personality you like? Save it as a preset and share the JSON file. Others can drag-and-drop it into their Schism to use your exact configuration.
